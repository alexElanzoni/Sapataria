# 👞 Sistema de Gestão para Sapataria

Sistema completo para gerenciamento de ordens de serviço de uma sapataria, desenvolvido com **HTML, CSS e JavaScript puro**.

O projeto funciona **100% offline**, sem necessidade de internet, servidor ou banco de dados externo, utilizando apenas **LocalStorage** para persistência dos dados.

---

# 🚀 Funcionalidades

## 👥 Clientes

* Cadastro de clientes
* Busca por nome e telefone
* Histórico completo de ordens de serviço

## 📦 Ordem de Serviço

* Cadastro de múltiplos itens por OS
* Tipos de item:

  * Sapato
  * Bolsa
  * Mala
  * Mochila
  * Bota
  * Outros
* Controle de quantidade
* Valor individual e total
* Características e observações
* Edição da OS mantendo o mesmo ID

## 📋 Painel do Dia

* Entregas do dia
* OS atrasadas com destaque visual
* Lista de ordens em aberto

## 💰 Caixa

* Entradas automáticas ao finalizar OS
* Registro manual de saídas
* Controle financeiro diário e mensal
* Métodos de pagamento

## 📊 Relatórios

* Relatório diário
* Relatório mensal
* Resumo financeiro
* Impressão otimizada

## 🖨️ Impressão

* Compatível com impressoras térmicas
* Suporte para:

  * 58mm
  * 80mm
* Layout preparado com `@media print`

# 🎯 Objetivo do Projeto

O objetivo deste sistema foi aplicar conhecimentos de desenvolvimento web em um problema real de um pequeno negócio.

Além da parte técnica, o foco também foi criar uma interface simples, rápida e funcional para uso no atendimento do dia a dia.

* configurar servidor
